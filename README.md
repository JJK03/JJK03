### 🛠️ Capabilities
현재 바로 실무에 적용하거나 개발이 가능한 기술 스택입니다.

- **Backend**: Java를 기반으로 한 **Spring Boot** 환경에서 REST API 및 비즈니스 로직 설계가 가능합니다.
- **Database**: **PostgreSQL/MySQL** 환경에서 데이터 모델링(ERD) 및 JPA를 활용한 효율적인 쿼리 작성이 가능합니다.
- **Tools**: **Git, Postman**를 활용하여 원활한 협업 및 개발 환경 구축이 가능합니다.

### 🌱 Currently Learning
더 깊이 있는 개발자가 되기 위해 집중적으로 파고들고 있는 분야입니다.

- **[Network Protocol]**: TCP/UDP를 넘어 WebSocket과 HTTP/3 등 실시간 통신 규격의 원리를 학습 중입니다.
- **[Data Security]**: AES, RSA 등 암호화 알고리즘의 실제 적용과 종단간 암호화(E2E) 시스템의 구조를 공부하고 있습니다.
- **[Spring Security]**: JWT 및 OAuth 2.0을 활용한 더 정교한 인증/인가 시스템 구축을 연구하고 있습니다.
